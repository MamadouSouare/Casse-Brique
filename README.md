# Casse-Briques Lua/LÖVE2D

Un jeu de casse-briques classique développé en Lua avec le framework LÖVE2D.

## Fonctionnalités

- **Menu Principal** : Point d'entrée simple et clair pour lancer une nouvelle partie.
- **Gameplay Classique** : Contrôlez une raquette pour renvoyer une balle et détruire toutes les briques.
- **Système de Vies** : Vous disposez de 3 vies. La partie se termine si toutes sont perdues.
- **Score** : Chaque brique détruite ajoute des points à votre score total.

## À Propos du Code

Ce projet a été l'occasion d'expérimenter avec les concepts de la Programmation Orientée Objet (POO) en Lua. L'architecture du jeu utilise des **métatables** pour simuler la création d'objets (comme les briques, la balle, la raquette) de manière simple et lisible, sans bibliothèque externe.
